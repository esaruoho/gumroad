# QA Media

Screenshots and videos for pull request documentation. Reference these in PR descriptions using raw GitHub URLs:

```markdown
![description](https://raw.githubusercontent.com/antiwork/gumroad/<branch>/qa-media/<filename>)
```

## Naming Convention

`pr-<number>-<description>.<ext>`

Examples:
- `pr-5160-pagination-page1.png`
- `pr-5160-pagination-page2.png`

## Cleanup

Media for merged/closed PRs can be removed periodically. Keep the directory itself.
